🎨 피그마 디자인 검수 도구

피그마 URL을 입력하면 자동으로 디자인을 분석하고 검수합니다

📋 피그마 파일 분석

피그마 파일의 URL과 액세스 토큰을 입력하여 디자인 요소들을 자동으로 분석할 수 있습니다.

💡 참고: 이 도구는 피그마 API를 사용하여 파일을 분석합니다. 액세스 토큰은 피그마 개발자 설정에서 생성할 수 있습니다.

🔧 사용 방법

  • 피그마 URL: 분석하고 싶은 피그마 파일의 공유 URL을 입력하세요
  • 액세스 토큰: 피그마 설정 → Account → Personal access tokens에서 생성
  • 분석이 완료되면 디자인 요소들과 텍스트 내용을 확인할 수 있습니다
🔍
자동 텍스트 추출
피그마 파일에서 모든 텍스트 요소를 자동으로 추출하고 분석합니다.
📊
상세 분석 보고서
디자인 요소의 구조와 텍스트 내용을 체계적으로 정리한 보고서를 제공합니다.
💾
보고서 다운로드
분석 결과를 HTML 형태로 다운로드하여 팀과 공유할 수 있습니다.

📖 기존 분석 결과 보기

이전에 분석된 결과들을 확인할 수 있습니다.

📋 디자인 검수 보고서

피그마 디자인 요소들의 상세한 검수 결과를 확인하세요.

📝 텍스트 검수 보고서

텍스트 요소들의 구현 상태와 매칭 결과를 확인하세요.

👥 팀 공유 페이지

팀원들과 분석 결과를 공유할 수 있는 페이지입니다.
